# 🧩 機能一覧・開発計画（feature_plan.md）

## 🎯 目的

このドキュメントは、Webアプリケーションの機能一覧と開発ステータスを整理し、今後の実装計画を明確にするためのものです。

---

## ✅ 実装済み機能

| 機能 | 概要 | ステータス |
|------|------|------------|
| 環境構築 | Djangoプロジェクト・アプリ作成、仮想環境・パッケージ管理 | 完了 |
| モデル設計 | Event, Artist, Performance, RelatedArtist, ManualEntry モデル定義 | 完了 |
| 管理画面 | モデル登録・編集・検索・フィルター機能 | 完了 |
| Spotify API連携 | アーティスト検索・保存、一括登録処理 | 完了 |
| 一覧表示 | アーティスト・イベントの一覧ページ（ListView） | 完了 |
| 詳細表示 | アーティスト・イベントの詳細ページ（DetailView） | 完了 |
| 検索機能 | アーティスト名による検索（クエリ or フォーム） | 完了 |
| イベント登録フォーム | JSによる日付選択付きフォーム | 完了 |

## 🚧 実装予定機能

| 機能 | 概要 | 優先度 | 予定時期 |
|------|------|--------|----------|
| 類似アーティスト取得 | Spotify `/related-artists` を使った予測ロジック | 高 | 11月中旬 |
| 出演予測ロジック | 類似度・日程・ジャンルなどをもとに出演可能性を推定 | 高 | 11月中旬〜下旬 |
| プレイリスト作成 | アーティストの人気曲を集めて予習用プレイリスト生成 | 中 | 11月下旬 |
| UI改善 | Bootstrap導入、検索結果の表示調整、レスポンシブ対応 | 中 | 随時 |
| テスト体制構築 | `pytest` やモックによるAPIテスト | 中 | 11月下旬 |
| README英語版 | 英語話者向けのREADME作成 | 低 | 必要に応じて |
