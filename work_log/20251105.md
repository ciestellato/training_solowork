## 🎯 目的（プロジェクトの狙い）

- **音楽フェスティバル管理システムの構築**
  - アーティスト情報の登録・一覧・詳細表示
  - イベントの作成・日程管理・出演者登録
  - Spotify APIとの連携によるアーティスト情報の自動取得
  - 管理者向けの登録フォームとバリデーション機能
  - 開発者・チーム向けにドキュメントとテスト体制を整備

---

## 🏗️ 構成（技術スタックと設計）

### 🔧 技術スタック
| 分類 | 使用技術 |
|------|----------|
| フレームワーク | Django 5.2.7 |
| テスト | pytest, pytest-django |
| API連携 | Spotify Web API（requests使用） |
| フロント | Bootstrap 5（CDN） |
| 環境管理 | python-dotenv |
| バージョン管理 | Git + GitHub |

### 📁 ディレクトリ構成（主要ファイル）
```
festival/
├── models.py         # Artist, Event, EventDay, Performance
├── views.py          # 一覧・詳細・登録処理

├── urls.py           # 名前空間付きURL定義

docs/
├── screen_flow.md    # 画面遷移図

├── feature_plan.md   # 機能計画
```
